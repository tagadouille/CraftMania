# Craftmania

# Prérequis :

Avoir une version de java supérieure à 21.

# Lancer le jeu :

Exécuter la commande suivante :

```bash
./gradlew run
```

# Comment jouer au jeu :

### Déplacements :

- Utiliser les touches : ZQSD
- Cliquer sur une case vide, le joueur s'y déplacera automatiquement

On peut intéragir avec un élément de la carte en étant à côté puis en cliquant dessus.

### Intéraction possible sur un élément via un clique: 

- Récolter une resource
- Utiliser le marché pour vendre ou acheter des ressources
- Aller sur l'interface d'une machine pour récupérer ce qu'elle a produit
